MY E-COMMERCE WEBSITE

This page contains the case study of My E-commerce Website which includes the Project Overview, Tools Used and Live Links to the official product.

Project Overview

My E-commerce Website is a website project that tries to look like a real e-commerce website, which consumes a API called "fakestoreapi.com", exibing his .json format result and doing operations with it, such as filtering, queries, deletions, etc.

My E-commerce Website was idealized and designed by me. I took inspirations on most of the famous e-commerce websites in the world, such as Ebay, Walmart, Amazon, etc.
